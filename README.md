# MemoryVault

A Digital Time Capsule Social Platform where users can create, store, and unlock memories in the future.

---

## Live Website

🌐 https://memoryvault-oitk.onrender.com/

---

## Project Overview

MemoryVault allows users to:

- Create digital time capsules
- Store future messages and memories
- Set unlock dates for capsules
- Manage personal memories securely
- Access a personalized dashboard

The platform is designed to preserve memories digitally and reveal them at meaningful future moments.
